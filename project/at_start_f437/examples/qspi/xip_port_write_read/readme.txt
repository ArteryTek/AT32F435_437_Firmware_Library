  this demo is based on the at-start board, in this demo, systick used for 
  delay function. qspi xip port operate write/read, if qspi test pass, led3
  fresh, else led2 fresh.
  the qspi ram is ly68l6400. 
  - qspi io0   --->   pf8
  - qspi io1   --->   pf9
  - qspi io2   --->   pf7
  - qspi io3   --->   pf6
  - qspi sck   --->   pf10
  - qspi cs    --->   pg6
