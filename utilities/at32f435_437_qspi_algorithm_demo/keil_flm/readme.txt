  this demo is based on the at-start board. qspi flash programming algorithms
  is showed in this demo.  the target flash is esmt32m. 
  when choose qspi1_algorithm demo,the following pin need tobe select
  by Configuration Wizard:
  - QSPI1 CS pin	: PB6/PB9/PB10/PC11/PG6
  - QSPI1 SCK pin	: PB1/PB2/PD3/PF10
  - QSPI1 IO0 pin	: PA6/PB0/PB11/PC9/PF8/PH2
  - QSPI1 IO1 pin	: PA7/PB10/PF9/PH3
  - QSPI1 IO2 pin	: PA15/PC4/PC8/PE2/PF7/PG9
  - QSPI1 IO3 pin	: PA1/PB3/PD13/PF6/PG14
  when choose qspi2_algorithm demo,the following pin need tobe select
  by Configuration Wizard:
  - QSPI2 CS pin	: PB8/PG8
  - QSPI2 SCK pin	: PB1/PG13
  - QSPI2 IO0 pin	: PB0/PE7/PG14
  - QSPI2 IO1 pin	: PB7/PC3/PE8/PG12
  - QSPI2 IO2 pin	: PA5/PE9/PG10
  - QSPI2 IO3 pin	: PA3/PE10/PG11
